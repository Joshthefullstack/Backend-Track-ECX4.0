## GUIDE
* Download the Repo
* run npm install
* make sure your mongodb server is up and running 
* get the connection string for your mongodb server
* create a .env file
* fill out the MONGO_URL string e.g MONGO_URL="mongodburl" [replace mongodburl with the connection string]
* run npm start
